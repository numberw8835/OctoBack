OctoBack

🐙 Octo

Octo is a lightweight CLI backup manager that separates the "intent" (what to back up) from the "action"
(the actual backup process). By maintaining a curated index of files and folders, Octo ensures your
backup vault remains organized and predictable.

🚀 Usage
1. Setup

To begin, initialize your environment. This creates the octo.yaml configuration file.
~$ octo init

This allows you to define your storage locations, choose your backend tools, and set security
preferences.

Configuration ( octo.yaml )
The configuration is now organized by category to allow for easier expansion:
version: 1.0
storage:
vault_path: "/path/to/backup_vault"
index_path: "/path/to/index.json"
engine:
copy_tool: "rsync"

# Options: rsync, cp

compression: "zstd"

# Options: none, gzip, zstd

compression_level: 3

# 1 (fastest) to 22 (best)

security:
encryption:
enabled: false
algorithm: "AES-256"
system:
os_type: "linux"
track_history: true

2. The Indexing Phase
Instead of backing up folders on the fly, Octo uses an "Index." You tell Octo which folders are important,
and it remembers them for future backup runs.

Add a folder to the index
~$ octo add folder/

This adds the specified path to the index. Octo will now track this folder for every subsequent backup.

Recursive Indexing
~$ octo add -R folder/

The -R flag recursively scans the directory and adds all sub-folders and files to the index individually,
allowing for more granular control during restoration.

3. The Execution Phase
Once your index is set, you can trigger the actual data transfer.

Run Backup
~$ octo backup

Octo reads the index.json , applies the compression and encryption settings from your config, and
syncs the data into the vault while maintaining the original directory structure.

4. The Recovery Phase
Restoring data is handled via the restore command, which fetches data from the vault based on your
index.

Restore a specific folder

~$ octo restore folder/

Octo looks up the path in the index and restores the most recent version from the vault to your current
directory.

Full System Restore
~$ octo restore --all

Restores all indexed folders and files currently stored in the backup vault.
